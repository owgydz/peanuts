Peanuts is a fun Python-based programming language with its own syntax and runtime, modules, and more.

## Example

```peanuts
import nutmath

nut main():
    jar x = nutmath.add(5, 3)
    print(x)

main()
```

Run it with:
`py peanuts.py run <file>.nut`

The output should be:
`peanut_lang_output: 8`
